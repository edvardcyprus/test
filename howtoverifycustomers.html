Account Verification

When Payments and antifraud department is not in the office and you have completed all the tasks, proceed with customer verification.

To verify a customer:

1. Locate initial request/email which includes the documents

2. Open the “Public” Folder – REGISTERS – New accounts register (ver. 2021)

3. Check if there is a folder in the Public network created already or create a new folder in the Public network. And name it as the customers account number, who wrote a request for identification.

4. Download, save and rename the attachments to the folder accordingly:

Passport

ID Front

ID Back

Proof of Address

4. Open the “Public” Folder – REGISTERS – New accounts register (ver. 2021)
